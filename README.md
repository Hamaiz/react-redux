# Files FrontEnd

This is the website which is the frontend of a [Golang Backend](https://github.com/Hamaiz/rest-api-golang) . This forntend is written with ReactJS, NextJS and Redux. It is a Q/A site. **This project is still under development.**

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. You have to have NodeJs installed on your machine.

### Installing

1. Clone the repo

```
git clone https://github.com/Hamaiz/react-redux.git
```

2. Run npm install inside project root directory

```
npm install
```

3. Create .env.local

```
touch .env.local
```

4. Add all the Enviornment variables
   - NEXT_PUBLIC_URL - (this is the backend url)
5. Start the server

```
npm run dev
```

6. Explore

```
Enjoy! :)
```
